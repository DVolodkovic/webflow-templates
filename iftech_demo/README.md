# Iftech - Next.js Website

Modern Next.js implementation of the Iftech Webflow template with full animations and interactions.

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **React 18**
- **Framer Motion** (for animations)
- **Webflow JavaScript** (for original animations)

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
npm start
```

## Project Structure

```
├── app/
│   ├── components/      # React components
│   ├── globals.css      # Global styles
│   ├── layout.tsx        # Root layout
│   └── page.tsx          # Home page
├── public/
│   └── assets/           # Static assets (CSS, JS, images)
└── package.json
```
